## Foundations

Space begins at :: **100 km** above mean sea level (the Kármán line)

The three unique military advantages of space :: **Persistence** (repeated passes), **Global Reach** (constellation covers all Earth), **Speed** (EM signals at light speed)

USSF was established on :: **20 December 2019** as the sixth branch of US armed services

USSF members are called :: **Guardians** (not Airmen — the distinction matters in person)

The difference between USSF and USSPACECOM ::
?
**USSF** = service branch (organise, train, equip forces)
**USSPACECOM** = combatant command (employs forces operationally)
Think of it like the US Army vs US Army Forces Command

SSA was redesignated SDA because ::
?
**Space Situational Awareness → Space Domain Awareness**
The redesignation signals that space is now an **actively contested warfighting domain** — adversaries are deliberately trying to deny, disrupt, degrade, and destroy capabilities. Passive monitoring was no longer sufficient.

## Orbital Mechanics

The three orbital regimes and their altitude ranges ::
?
**LEO** — 160 to 2,000 km (ISS, SDA PWSA, Space Fence coverage)
**MEO** — 2,000 to 35,786 km (GPS, GLONASS, Galileo)
**GEO** — 35,786 km (WGS, AEHF, MUOS, GSSAP)

LEO satellite velocity :: ~**7.8 km/s** (28,000 km/h) — in continuous freefall, forward velocity balances gravity

Why GEO is easier to jam than LEO :: GEO satellites are **stationary relative to Earth** — fixed geometry enables a directional jammer to aim once and stay on target. LEO satellites move rapidly across the sky.

RPO stands for :: **Rendezvous and Proximity Operations** — a satellite manoeuvring close to another for intelligence, pre-positioning, or attack

## Satellite Capabilities

The six core military satellite capability areas ::
?
1. SATCOM (command and communications)
2. PNT / Navigation (GPS)
3. Missile Warning and Tracking
4. ISR / Intelligence
5. Environmental Monitoring / Space Weather
6. Space Domain Awareness

WGS's critical vulnerability :: **No anti-jam protection** — operates in unprotected X/Ka band; a directional jammer can deny it without any space capability

WGS stands for :: **Wideband Global SATCOM** — 10 GEO satellites, X/Ka band, high data rate

Australia's WGS contribution :: Funded **WGS-6** at A$927M in 2007; launched August 2013; provides global access until ~**2029**

AEHF stands for :: **Advanced Extremely High Frequency** — 6 GEO satellites, anti-jam, nuclear survivable, restricted to NCA and select users

MUOS stands for :: **Mobile User Objective System** — 5 GEO satellites, 3G-like UHF SATCOM, Australia hosts one of **four** ground stations globally (Kojarena WA)

GPS L1/L2/L5 — which is military-only ::
?
**No signal is military-only** at L1/L2/L5 — the military advantage is **M-Code** (a military-only signal overlay on L1 and L2), which requires a specific military-grade receiver and provides better anti-jam performance

## Space Domain Awareness

The four core SDA functions ::
?
1. **Detect / Track / Identify** (D/T/ID)
2. **Characterisation** (mission, behaviour, attribution)
3. **Threat Warning and Assessment**
4. **C2 Integration** (fuse into actionable products)

Space Fence location and capability :: **Kwajalein Atoll** — S-band phased array, detects **marble-sized** objects in LEO

GEODSS limitation :: **Night and clear weather only** — ground-based electro-optical; degraded by cloud and daylight

GSSAP stands for :: **Geosynchronous Space Situational Awareness Program** — 6 near-GEO inspection satellites providing close-approach characterisation of GEO belt

ATLAS replaced :: **SPADOC** (Space Defense Operations Center) — 1980s-era legacy system; ATLAS declared operationally accepted **September 2025**

18th SDS role :: Maintains the **space object catalogue** and issues conjunction warnings globally — operates from Vandenberg SFB under Delta 2

SST Exmouth's strategic value :: Provides **unique southern-sky coverage** of the deep-space GEO belt — fills a gap no other sensor in the network covers

## Counterspace Threats

The five counterspace threat categories ::
?
1. **Kinetic DA-ASAT** — missiles physically destroy satellites; creates debris
2. **Non-Kinetic Directed Energy** — lasers dazzle/blind; HPM damages electronics
3. **Electronic Warfare** — jamming and spoofing
4. **Cyber** — attacks on ground stations, links, software
5. **Co-orbital RPO** — manoeuvre near target for intelligence or attack

China's 2007 ASAT test target and result :: Destroyed **Fengyun-1C** weather satellite — created **3,000+ debris fragments** still in orbit today

Russia's 2021 ASAT test :: Destroyed **Cosmos 1408** — created 1,500+ fragments, threatened ISS crew (including Russian cosmonauts)

US DA-ASAT moratorium :: April 2022 — US committed **not to conduct destructive DA-ASAT testing**

Most frequently employed counterspace method :: **Electronic Warfare** (jamming/spoofing) — cheap, deniable, reversible; occurs daily in conflict zones

Krasukha-4 capability :: Russian ground-based EW system — reportedly suppresses drone reconnaissance at **150–300 km** range

## GNSS

The four GNSS constellations ::
?
**GPS** (US) — 31 satellites, MEO 20,200 km, L-band, M-Code military signal
**GLONASS** (Russia) — 24 satellites, MEO 19,100 km, FDMA architecture
**BeiDou** (China) — 35+ satellites, MEO + GEO + IGSO, B3A anti-jam + SMC datalink
**Galileo** (EU) — 30 satellites, MEO 23,222 km, high-accuracy civilian

BeiDou SMC capability :: **Short Message Communication** — a two-way datalink embedded in the B3A navigation signal; enables passing targeting coordinates via GNSS. GPS does not have this.

Why Shahed-136's GNSS choice matters :: Uses **GLONASS** (not GPS) — US GPS jammers have **zero effect** on GLONASS-guided munitions. Counter-drone planning must identify which constellation the threat uses.

CRPA stands for :: **Controlled Reception Pattern Antenna** — hardware anti-jam solution using multiple antenna elements to null jamming signals

## SATCOM Jamming

The four satellite jamming techniques ::
?
1. **Uplink jamming** — jam the signal from ground to satellite
2. **Downlink jamming** — jam the signal from satellite to ground terminal
3. **Bent-pipe jamming** — inject a false signal that the satellite retransmits
4. **On-orbit jamming** — a co-orbital satellite jams at the space segment

Why downlink jamming is the easiest counterspace technique :: Only needs to be **near the ground terminal** — not near the satellite. A vehicle-mounted jammer near an antenna can deny comms without any space capability.

## Tactical Data Links

Link 16 frequency and access method :: **L-band 960–1215 MHz**, **TDMA** (128 time slots/second), frequency hopping across 51 frequencies

Link 16's key vulnerability :: L-band emissions are **detectable by adversary ESM** — enables direction-finding, platform tracking, order-of-battle analysis. This is why F-22 was receive-only and F-35 uses MADL operationally.

Link 16 BLOS options ::
?
1. **JREAP** — Link 16 over SATCOM relay (adds latency, SATCOM-dependent)
2. **SDA Transport Layer (PWSA)** — Link 16 from LEO space (no new terminals needed)
3. **Link 22** — separate HF/UHF link for BLOS (SATCOM-independent)

MADL stands for :: **Multifunction Advanced Data Link** — F-35-only LPI/LPD directional link; cannot directly share with non-F-35 platforms

CEC vs Link 16 :: CEC provides **fire-control quality tracks** (not just surveillance quality) — a ship can launch a missile guided by another ship's radar. Link 16 provides a shared picture but not fire-control quality data.

First time Link 16 was relayed from space :: **November 2023** by SDA Tranche 0 — existing Link 16 radios work with the space layer without modification

JREAP limitation :: Depends on **SATCOM availability** — in a SATCOM-contested environment, JREAP connectivity is lost. Also adds **latency** that degrades time-critical tracks.

## Coalition and ADF

Operation Olympic Defender purpose :: Multinational framework for **combined space operations** — led by the US, includes AU, UK, CA, NZ, DE, FR and others; provides a structure for sharing SDA and coordinating space activities

Australia's four key space contributions ::
?
1. **SST Exmouth** — C-band space surveillance telescope, southern sky coverage
2. **Kojarena** — MUOS Radio Access Facility (one of only four globally)
3. **WGS-6** — funded at A$927M, provides global access until ~2029
4. **Pine Gap** — strategic intelligence facility (missile warning, SIGINT, satellite tracking)

AUSSpOC role :: **Australian Space Operations Centre** — ADF embed at CSpOC (Vandenberg SFB); the interface between ADF and the combined space operations picture

Op DYURRA :: Australia's named **contribution to coalition space operations** — the operational name to know for planning and reporting

## Kill Chains

F2T2EA stands for :: **Find, Fix, Track, Target, Engage, Assess** — the kill chain framework

Which kill chain phase does GPS primarily support :: **Target** (delivering GPS coordinates to weapon) and **Engage** (in-flight guidance)

China's anti-ship kill chain primary sensor :: **Yaogan satellites** (SIGINT and electro-optical) + over-the-horizon radar for initial targeting of carrier groups

Iran's Shahed-136 navigation layers ::
?
1. **GLONASS** primary (GPS jamming doesn't affect it)
2. **INS** backup
3. **Terrain matching** (TERCOM-like) for terminal phase
Layered architecture provides resilience across jamming scenarios
