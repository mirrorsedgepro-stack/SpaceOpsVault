# Local Document Library

> [!info] What This Note Is For
> This note is the central hub for linking locally-saved copies of primary source documents to the vault. Use the **Better File Link** plugin (`Ctrl+P` → **Add file link**) to attach each document. Once linked, you can click through to the file without leaving Obsidian.
>
> **First time?** → [[Plugin Installation Guide#File Linking Plugin]] — install Better File Link before adding any links here.

> [!tip] Recommended Folder Structure
> Save your documents in a predictable location so links remain valid if you move between machines. A subfolder next to the vault works well:
> ```
> SpaceOpsVault/
> └── _docs/
>     ├── doctrine/
>     ├── threat-assessments/
>     └── adf/
> ```
